# Archived Python reference model

Status: archived historical evidence; no active source, build, test, workflow,
or tool depends on this directory.

This model consumes a private Python dictionary AST and tests its own static
checker against a separate bounded interpreter. It does not parse Whitefoot or
exercise and compare with the Rust compiler. The owner retired it from the
active repository gate and language-change workflow on 2026-07-22.

The preserved files remain useful only for understanding the historical model.
They do not define current language behavior or compiler requirements.
