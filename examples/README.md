# Examples

Dogfood and example Whitefoot programs. This directory grows as the compiler
becomes able to resolve, check, and run real programs (see `docs/roadmap.md`,
Phase 7 onward). Until then it is intentionally a placeholder: production-shaped
programs (binary/compression, text and command-line processing, graph-shaped
work, one sustained workload) land here to expose missing language and compiler
capabilities.
